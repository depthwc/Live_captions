# Live_captions
live caption

Its an app that showes live captions of the audio in system for sapareted apps. u can choose which app to enable captions for. 

it shows captions for all the apps at the same time, the GUI is sapareted for each app. GUI is transparent and can be moved anywhere on the screen. and its customaziable. making it easy to use for people who need captions while watching videos or listening to audio at the same time.
